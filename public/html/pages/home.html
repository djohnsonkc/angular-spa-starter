So, you wanna build an SPA?

This app is a starter project for creating an SPA (single-page app) using AngularJS, which includes a nice 'scaleUp' animation for view changes. Node.js is used to serve the main index.html page and is also used to provide Gzip compression and enable client-side, browser caching of assets to clients making requests to the app.
